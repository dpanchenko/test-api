# Test API

## Environment

 - Development environment
`Node v12.13.1`
`MongoDB v4.0.11`

## Start

 - Clone or download this repository
``` bash
git clone
```

 - Enter your local directory, and install dependencies:

``` bash
npm i
```

## Commands

``` bash
# run server in development mode
npm run dev
```

``` bash
# run linter
npm run lint
```

``` bash
# run tests
npm test
```
